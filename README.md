# URL Shortener
Simple url shortener on flask.

# Install
```bash
> git clone https://github.com/Good5263/url-shortener.git
> cd url-shortener
> pip install virtualenv
> py -m virtualenv .env
> .env\Scripts\activate
> pip install -r requirements.txt
> deactivate
```

# Launch
```bash
> .env\Scripts\activate
> python app.py
```

# Exit
```bash
> Ctrl+C  
```
